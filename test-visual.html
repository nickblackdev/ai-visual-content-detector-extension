Expected Detection Results:

  • AI-Generated Content: Should show higher AI probability scores due to:
    • Perfect square dimensions (512x512, 1024x1024, 1536x1536)
    • Power-of-2 dimensions (512, 1024, 1536)
    • Sample URLs with "random" parameters
    • Standard AI generation aspect ratios
  • Human-Captured Content: Should show lower AI probability scores due to:
    • Non-standard dimensions (4032x3024, 3024x4032)
    • Natural aspect ratios (4:3, 16:9)
    • Real-world camera dimensions
    • Varied aspect ratios

Detection Algorithm Features:

  • Perfect Symmetry: Detects perfect squares and power-of-2 dimensions
  • Unusual Patterns: Identifies common AI generation dimensions and aspect ratios
  • Color Consistency: Analyzes image characteristics for AI patterns
  • Texture Analysis: Checks for high-resolution and perfect square dimensions
  • Edge Detection: Identifies common AI generation sizes
  • Metadata Analysis: Checks URLs for AI generation indicators
  • File Characteristics: Analyzes dimensions and aspect ratios
